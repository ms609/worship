Song Synchronizer

This page displays:

  • Any songs where there is a difference between the local file and the file stored on the server.
  • Any set lists present on the server or the local machine.

To use the page:

  • For each song, click on the version that you want:
    • Select "On this PC" version to send your version to the server, so that it's available to others
    • Select "On server" to retrieve the latest version of the song from the server
  • Use the Synchronize button below to enact your changes.
